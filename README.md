# Belo Zero — Official Website

Official artist website for **Belo Zero of Do or Die** — Chicago's West Side Legend.

## Tech
- Pure HTML/CSS/JS — no frameworks, no build step
- Self-contained single file (`index.html`)
- YouTube click-to-play thumbnails
- Sticky music player (bottom bar)
- Live product links to shop.wickedentertainment.net

## Assets
- `index.html` — full website
- `do_or_die_logo.jpg` — Do or Die logo used in header & hero

## To go live
Upload `index.html` and `do_or_die_logo.jpg` to any web host (Netlify, GitHub Pages, etc.)
